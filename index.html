Hakkımda

Merhaba. Kod yazmak çok keyif aldığım bir iş. Arduino, C# ve Python ile programlar yazabiliyorum.
Kendimi Web konusunda da geliştirmek için Patika ve Kodluyoruz sitelerinde verilen eğitimlerden yararlanıyorum. Size de tavsiye ederim.

Bazı Yazılım Çalışmalarım

  1. Arduino

    • Web entregeli (BackEnd only) - Isı-Nem Takip sistemi

      Bu çalışmamda ESP8266 Wifi modülünü ve DHT22 ısı-nem sensörünü kullanarak
      yaptığım cihaz sayesinde ölçülen değerlerin oluşturduğum web sitesine iletilmesini gerçekleştirdim.

      Bu sitede hesap oluşturma, e-posta adresine gönderilen kod ile hesap doğrulama, şifremi unuttum, hesaba yeni cihaz kayıt etme,
      kayıtlı cihazların listelenmesi ve ölçümlerinin gösterilmesi gibi sayfalar/fonksiyonlar mevcut.

      Ancak bu çalışmamda öncelikli amacım arduino ile web arasında veri iletişimini sağlamak olduğundan
      ve mevcut Front End bilgim yeterli olmadığından site prototip aşamasında.

    2. Arduino, Mikrodenetleyici veya Mikroişlemciler ile yazılım geliştirmeyi kolaylaştıran bir devredir.

  2. C#

    • Otomasyon Programi

      Bir Nesne-İlişkisel eşleme çerçevesi (object relational mapper (O/RM)) olan Entity Framework 6 kullanılarak Code First yöntemi
      ile yaptığım bu otomasyon programı FrontOffice ve BackOffice olmak üzere 2 Ana Modülden oluşmakta.

      BackOffice Modülü İşletmenin kendi içerisinde yaptığı değişikliklerin sisteme girildiği modüldür.
      Admin tarafından yapılan Yeni Ürün veya Personel Ekleme gibi işlemler bulunur.

      FrontOffice ise İşletmenin çalıştığı esnada yapılan işlemlerin gerçekleştiği modüldür.
      Siparişlerin alınması, üretilen ürünlerin adetleriyle sisteme girilmesi gibi personel tarafından yapılan işlemleri bulunur.

    • Web entregeli (BackEnd only) - Isı-Nem Takip sistemi

      Arduino kısmında bahsettiğim çalışmanın Web sayfasındaki Back End fonksiyonlarini C# (ASP.NET) ile kodladim.

  3. Python

    • Basit Web tarayıcı

      PyQt kütüphaneleri ile yaptığım, Önceki sayfa, sonraki sayfa, sayfa yenileme, adres çubuğu ve home sayfası
      gibi temel unsurların bulunduğu basit bir web tarayıcı uygulaması.

    • Anlık olarak küçük çaplı programlara ihtiyaç duyduğumda

      Python dilinde çalışmak diğerlerine göre daha hızlı olduğundan (IDE nin açılması gibi), kisa çözümü olan ihtiyaçlarda kullandığım bir dil.
      Bir benzetme olarak, Kitap yazdığımda C#, kisa not almam gerektiğinde ise python tercih ediyorum.

      Örnek olarak: Toplu dosya tipi dönüşümü, toplu yeniden adlandırma veya PyAutoGui işlemleri gibi aslında manuel yapılabilen
      ama sayısı çok olduğunda python sayesinde hizlandirilabilinen işlemler.
